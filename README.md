## Aegis-Lam Lite v1.0.0-beta (Global Integrated)



Aegis-Lam is an intelligent, lightweight security sentinel designed to bridge the security gaps in remote work environments. This release marks the first global integration, offering seamless protection across different locales.



###  Network Configuration

For stable communication and telemetry, this version uses the following fixed ports. Please ensure these are not blocked by your local firewall:

* **Fixed Ports**: `8443`, `8080`, `9000`



###  Key Features

* **Global Language Support**: Automatically detects system locale (KOR/EN) to provide a native user experience.

* **Real-time Process Monitoring**: Advanced heuristic engine to detect and mitigate unauthorized activities.

* **Anti-Debugging & Tamper Protection**: Built-in logic to prevent reverse engineering of the security core.

* **Crash-Resistant Architecture**: Optimized for stability with automated recovery protocols.



###  Aegis-Lam PRO (Upcoming)

We are currently developing the **PRO Version** for advanced security requirements.

* **Zeroing Function**: The PRO version will include a 'Zeroing' feature to completely neutralize identified threats and wipe malicious traces.

* **Advanced Console**: Enhanced management capabilities for enterprise-level security.



###  Data Policy

* **Log Collection**: This software collects activity logs to identify system vulnerabilities and improve bug fixes. 

* All collected data is used strictly for enhancing the security logic and stabilizing the Aegis-Lam engine.
